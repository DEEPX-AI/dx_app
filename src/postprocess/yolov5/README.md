# yolov5

This module provides post-processing for YOLOv5 object detection models. It extracts bounding boxes, class labels, and confidence scores from model outputs.

- **yolov5_postprocess.cpp/h**: Implements YOLOv5 post-processing logic.
- Suitable for general object detection tasks.

## Usage
Include the header and link the compiled library in your project to use YOLOv5 post-processing functions.

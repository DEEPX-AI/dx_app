# yolov5face

This module provides post-processing for YOLOv5Face models, specialized for face detection and landmark extraction.

- **yolov5face_postprocess.cpp/h**: Implements post-processing for face detection and facial landmarks.
- Use with YOLOv5Face model outputs.

## Usage
Include the header and link the compiled library in your project to use YOLOv5Face post-processing functions.

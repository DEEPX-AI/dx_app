# yolox

This module provides post-processing for YOLOX object detection models. It extracts bounding boxes, class labels, and confidence scores from model outputs.

- **yolox_postprocess.cpp/h**: Implements YOLOX post-processing logic.
- Suitable for general object detection tasks.

## Usage
Include the header and link the compiled library in your project to use YOLOX post-processing functions.

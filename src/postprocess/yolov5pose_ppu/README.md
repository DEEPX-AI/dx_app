# yolov5pose

This module provides post-processing for YOLOv5Pose models, extracting human pose keypoints from model outputs.

- **yolov5pose_postprocess.cpp/h**: Implements post-processing for pose estimation.
- Use with YOLOv5Pose model outputs.

## Usage
Include the header and link the compiled library in your project to use YOLOv5Pose post-processing functions.

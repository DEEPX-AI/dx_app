# yolov9

This module provides post-processing for YOLOv9 object detection models. It extracts bounding boxes, class labels, and confidence scores from model outputs.

- **yolov9_postprocess.cpp/h**: Implements YOLOv9 post-processing logic.
- Suitable for general object detection tasks.

## Usage
Include the header and link the compiled library in your project to use YOLOv9 post-processing functions.

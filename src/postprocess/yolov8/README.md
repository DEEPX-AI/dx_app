# yolov8

This module provides post-processing for YOLOv8 object detection models. It extracts bounding boxes, class labels, and confidence scores from model outputs.

- **yolov8_postprocess.cpp/h**: Implements YOLOv8 post-processing logic.
- Suitable for general object detection tasks.

## Usage
Include the header and link the compiled library in your project to use YOLOv8 post-processing functions.
